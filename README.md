# TODO UI and BACKEND app

## **UI** 
### Technologies:
* React;
* Redux;
* TypeScript;
* Ant Design.

## **BACKEND**
### Technologies:
* Java 11;
* Spring Boot 2.3.0;
* Swagger;
* RxJava;
* Postgres;
* Maven.

### To run backend:
1. Navigate to the main TODO directory.
2. Run build.sh with super user rights (using bash terminal).
